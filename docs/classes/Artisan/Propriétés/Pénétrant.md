*Propriété d'Arme Légendaire*
**Composantes :** Arme exotique chef-d'œuvre avec la propriété [[Propriétés d'Arme#Munitions|Munitions]] ou [[Propriétés d'Arme#Jet|Jet]] qui inflige des dégâts perçants.

Cette arme transperce ses cibles. Lorsque vous réalisez une attaque à distance avec cette arme et que vous n'avez pas le désavantage, vous pouvez cibler toutes les créatures dans une ligne droite dans la portée normale de l'arme. Vous n'effectuez pas de jet d'attaque pour cette attaque. À la place, chaque créature dans la ligne doit effectuer un jet de sauvegarde de Dextérité ou subir les dégâts comme si elle avait été touchée.
