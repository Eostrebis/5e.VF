Vous apprenez le cantrip [[moquerie cruelle]]. De plus, lorsqu'une créature que vous voyez effectue un jet d'attaque contre vous, vous pouvez utiliser votre réaction pour lancer [[moquerie cruelle]], ciblant l'attaquant. Si la créature échoue son jet de sauvegarde, elle obtient le désavantage sur le jet d'attaque déclencheur, plutôt que sur le prochain jet d'attaque.

Une fois que vous avez lancé ce cantrip comme réaction, vous ne pouvez le refaire jusqu'à ce que vous terminiez un repos.
