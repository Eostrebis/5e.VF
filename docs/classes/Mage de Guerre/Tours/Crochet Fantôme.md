_Prérequis : cantrip [[Grappin Fantôme]]_

Vous pouvez lancer le cantrip [[grappin fantôme]] pour une action bonus. Si vous le faites, sa portée est réduite à 15 ft. Lorsque vous lancez le sort pour une action, la cible est attirée vers vous de 10 ft. supplémentaires.
