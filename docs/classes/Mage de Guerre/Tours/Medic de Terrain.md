_Prérequis : Maison des Fous_

Vous apprenez le cantrip [[stabilisation]], qui ne compte pas dans le nombre de cantrips connus. De plus, lorsque vous lancez [[stabilisation]] sur une créature avec 0 point de vie, la créature regagne 1 point de vie ainsi qu'un nombre de points de vie temporaires égal à votre niveau de mage de guerre, qui persistent pendant 1 minute. Une fois qu'une créature a regagné des points de vie avec ce tour, elle ne peut le refaire avant d'avoir terminé un repos long.
