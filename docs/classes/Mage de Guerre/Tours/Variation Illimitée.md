_Prérequis : cantrip [[Prestidigitation]]_

Vous êtes devenu particulièrement compétent dans l'utilisation du cantrip [[prestidigitation]] pour imiter d'autres sorts. Lorsque vous lancez [[prestidigitation]], vous pouvez l'utiliser pour reproduire les effets d'un autre cantrip qui n'inflige pas de dégâts, même un qui ne se trouve pas parmi la liste de sorts du mage de guerre. Pour ce faire, vous devez réussir un jet d'Intelligence (Arcanes) DD 15, sinon le sort échoue. L'Intelligence est votre caractéristique d'incantation pour les cantrips lancés avec ce tour, et ils comptent comme des cantrips de mage de guerre pour vous.
