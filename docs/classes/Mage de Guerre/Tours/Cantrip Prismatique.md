Une fois à chacun de vos tours, lorsque vous infligez des dégâts de force avec un cantrip de mage de guerre, vous pouvez tirer un d6 dans la cible de Cantrip Prismatique afin de changer le type de dégâts pour cette incantation. Vous pouvez appliquer des tours normalement en utilisant le nouveau type de dégâts. De plus, vous pouvez appliquer le tour correspondant au type de dégâts, même si vous ne le connaissez pas.

Vous pouvez utiliser ce tour un nombre de fois égal à votre modificateur d'Intelligence, et vous récupérez les utilisations dépensées lorsque vous terminez un repos long.

| d6  | Type de Dégâts | Tour de Mage de Guerre |
| --- | -------------- | ---------------------- |
| 1   | Acide          | [[Cantrip Corrosif]]   |
| 2   | Froid          | [[Cantrip Glacé]]      |
| 3   | Feu            | [[Cantrip Explosif]]   |
| 4   | Foudre         | [[Cantrip Statique]]   |
| 5   | Poison         | [[Cantrip Aspirant]]   |
| 6   | Tonnerre       | [[Cantrip Silencieux]] |
